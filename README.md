# Superhero Travel Adventures API

Spring Boot REST API for managing travel adventures.

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database

## Running locally

```bash
./mvnw spring-boot:run

Endpoints

GET /traveladventures

GET /traveladventures/bycountry/{country}

GET /traveladventures/bystate?state=SP
